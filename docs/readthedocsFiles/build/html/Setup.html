Setting up Rigbox¶

Below is a walk-through for setting up your experiment rig after installing.

Configuring Paths¶

The first thing to set up are the paths. Rigbox uses the +dat package for determining where to things such as experiment data and configuration files are located. The paths are set by +dat.paths(). A template of this file can be found in docs\setup\paths_template.m. Running addRigboxPaths() should automatically copy this template to the +dat folder, otherwise copy it manually.

Configuring the hardware¶

The next step is to create a hardware file, which contains all rig specific hardware settings (device IDs, PsychToolbox parameteres, etc.) A step-by-step guide can be found in docs\setup\hardware_config
